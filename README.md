# ToDoList App

## Description
The ToDoList App is a simple yet powerful application designed to help users manage their tasks efficiently. This application allows users to create, view, edit, and delete tasks, while keeping the interface clean and easy to navigate.

## Features
- User-friendly interface to manage tasks
- Create, edit, and delete tasks
- Mark tasks as complete
- Local storage functionality to save tasks
- Responsive design compatible with multiple devices
